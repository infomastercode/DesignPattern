# DesignPattern

Design Pattern คือ แนวทางการเขียนโปรแกรมแบบมีแบบแผน เพื่อแก้ไขปัญหาๆหนึ่ง ทำให้รูปแบบ code มีความสวยงาม เป็นระเบียน ยืดหยุ่น มีมาตรฐาน ทำให้เวลาถ่ายงาน หรือรับงาน จะทำให้เข้าใจได้ง่าย เช่น class นี้ ใช้ pattern นี้ เราก็จะเข้าใจได้ง่าย

### แบ่งออกเป็น 3 หมวดหมู่

### Creational Patterns
- Abstract Factory
- Builder
- Factory Method
- Prototype
- Singleton

### Structural Patterns
- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Proxy

### Behavioral Patterns
- Chain of Resp.
- Command
- Interpreter
- Iterator
- Mediator
- Memento
- Observer
- State
- Strategy
- Template Method
- Visitor
